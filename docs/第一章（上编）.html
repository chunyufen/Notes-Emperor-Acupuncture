1  第一章：黃帝內針的淵源與傳承複習精要

1.1 一、 主要概念與理論

  1. 學術淵源（第一節）：
    • 理論根基：黃帝內針源自中醫經典《黃帝內經》，是純正的經絡辨證針法派系。
    • 核心特色：強調「簡、便、驗、廉」，具備易學高效、技術安全的特點。
    • 操作規範：內針對進針範圍有極其嚴格的限制——進針範圍嚴格限定在肘膝以下。除非急救，否則軀幹部位、頭部及肘膝以上區域均為禁針區。
    • 技術特點：不追求提插捻轉，不講究針感得氣，其療效追求「猶拔刺也，如雪污也」，即立竿見影的轉化。
  2. 傳承脈絡與修習要求（第二節）：
    • 傳承途徑：包含文字傳承（經典典籍）、口耳傳承（師徒相授）與直接傳承（心法領悟）。
    • 醫者素養（三要素）：
      • 信：確信內針的法脈與理路方計，這是修習的基石。
      • 誠：遵循天人合一，照章辦事，不隨意更改法度。
      • 明：對法理有透徹的把握。內針的技術極簡，但其理法至深，明理方能游刃有餘。
  3. 理論依據（第三節）：
    • 理法核心：源於《素問·陰陽應象大論》。
    • 核心口訣：內針的總綱在於「善用針者，從陰引陽，從陽引陰，以右治左，以左治右」以及「陽病治陰，陰病治陽」。這套理論決定了內針在臨床取穴時的「異位同氣」邏輯。

1.2 二、 醫療實務示例要點（理論引申）

雖然第一章以理論淵源為主，未列具體病案，但文中明確了臨床應遵循的「轉化觀」：

  • 非介入性干預：內針認為療效源於激發患者自身的「陰陽自和」。
  • 療效標竿：若取穴正確（同氣相求），患者的痛楚或功能障礙應在進針當下即刻緩解（立竿見影）。
  • 安全邊界：從事醫療實務時，必須嚴守「肘膝以下」的禁區。這種設計不僅是為了避開重要臟器，更是為了利用四肢末端最靈敏的「氣機窗口」。

1.3 三、 練習題核心考點

  1. 內針的定義與禁區：黃帝內針在一般臨床操作（非急救）中，絕對禁針的部位有哪些？（答案：肘膝以上區域、整個軀幹、頭部、患處病灶）。
  2. 進針操作特色：內針法脈中，對於提插捻轉與針感（得氣）的要求為何？（答案：不需人為補瀉，不用提插捻轉，不講求得氣與否）。
  3. 理論來源：黃帝內針的理論依據主要出自《黃帝內經》中的哪一篇文章？（答案：《素問·陰陽應象大論》）。
  4. 醫者素養：傳承黃帝內針需要把握的三個要點是什麼？（答案：信、誠、明）。

1.4 導讀：醫療實務建議

  1. 法理重於技術：內針的針刺手法極其簡單，45度斜刺即可。它真正的難點在於「辨經」與「定格」。你們未來的功夫要花在熟悉經絡巡行上。
  2. 建立「和平」的醫療觀：內針被稱為「和平的使者」，意指它不強攻病灶，而是透過末端的調整，引導全身陰陽回歸中道。在實務中，請務必保持心態的平和與專注。
  3. 守住法脈的「誠」：不要因為覺得好學就隨意發揮。內針的每一條法則（如左病右治、上病下取）都有其深刻的易理背景，照章辦事是你們通往進階之路的捷徑。
