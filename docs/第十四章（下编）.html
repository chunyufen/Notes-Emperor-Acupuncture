21  第十四章：足部

四肢末端與全身氣機巧妙聯繫。足部在黃帝內針的體系中，不僅是足三陰、足三陽經的起訖點，更是執行「上病下治」與「陰陽倒換」的核心陣地。這章建立完整的四肢投影觀；這章解釋如何在複雜的下肢痛症中，靈活運用三焦定位實現「隨手而應」。

21.1 一、 主要概念與理論

  1. 三焦定位的層次性（第一節）
    • 大環境定位：足部位於人體最下端，在整體三焦中明確屬於下焦（地部）。
    • 局部三焦劃分（關鍵投影邏輯）：
      • 天部（上焦）：踝部區域。
      • 人部（中焦）：膝部區域。
      • 地部（下焦）：胯部（臀/腹股溝）區域。
    • 操作邊界：踝部至膝部可視為上焦（對應天），膝部至胯部視為中焦（對應人），胯部以上視為下焦（對應地）。
  2. 循行經絡的路徑辨析（第二節）
    • 醫者必須精確掌握足部六經的巡行，這是尋找「同氣點」的唯一依憑：
      • 足太陽膀胱經：從腰中下夾脊，貫臀，入膕中，出外踝之後，循京骨至小趾外側。
      • 足陽明胃經：下髀關，抵伏兔，下膝臏中，下循脛外廉，入中指外間（2、3 趾間）。
      • 足少陽膽經：下循髀陽，出膝外廉，下外輔骨之前，直下抵絕骨之端，下出外踝之前，循足跗上，入小指次指之間（4、5 趾間）。
      • 足太陰脾經：起於大趾之端，循內側白肉際，過核骨後，上內踝前廉，上膝股內前廉，入腹。
      • 足少陰腎經：起於小趾之下，邪走足心（湧泉穴），出於然谷之下，循內踝之後，別入跟中，上股內後廉。
      • 足厥陰肝經：起於大趾叢毛之際，上循足跗上廉，去內踝一寸，上踝八寸，交出太陰之後，上股內廉。

21.2 二、 足部經絡同氣對應基準表（核心參考）

本表根據內針總則與各論內容彙整，是大家日後行醫定位的「指南針」：

經脈名稱 足部病灶對應區域 手部同氣（對應） 足部同氣（對應）
陽明經 膝臏、脛外廉、2/3 趾 合谷穴 / 曲池穴 陷谷穴 / 足三里穴
少陽經 膝外廉、外踝前、4/5 趾 中渚穴 / 支溝穴 足臨泣穴 / 陽陵泉穴
太陽經 膕中、外踝後、小趾側 後溪穴 / 小海穴 申脈穴 / 委中穴

21.3 三、 針治法則（第三節）

在醫療實務中，方位定格是「一針見效」的保證：

  1. 左病右治：症在左足，取右腕（肘）或右踝（膝）同氣。
  2. 右病左治：症在右足，取左腕（肘）或左踝（膝）同氣。
  3. 中間或雙側：按「男左女右」取穴，或以嚴重側為主證取其對側。

21.4 四、 醫案示例要點（第四節）

  1. 踝關節扭傷（天部/上焦證）：
    • 醫案：18 歲男性，右外踝扭傷痛至足跟。
    • 辨證：涉及陽明經、少陽經。病在上焦（天部）。
    • 方案：右病左治。取左手腕部（天部）。
    • 選穴：左手陽明經之陽溪穴、少陽經之陽池穴附近阿是點。
  2. 膝蓋疼痛（人部/中焦證）：
    • 醫案：65 歲女性，左膝前側及內側疼痛。
    • 辨證：涉及太陰經、厥陰經、陽明經。病在中焦（人部）。
    • 方案：左病右治。取右膝對應區域。
    • 選穴：右側陰陵泉穴（太陰）、曲泉穴（厥陰）、足三里穴附近阿是點。
  3. 坐骨神經痛（地部/下焦證 + 陰陽倒換）：
    • 醫案：50 歲女性，右臀部後外側及大腿外側酸痛。
    • 辨證：涉及少陽經。病在下焦（地部）。
    • 方案：右病左治。因膝以上禁針，執行陰陽倒換，取左手腕（天部）。
    • 選穴：左手少陽經外關穴附近刺入一針。

21.5 五、 練習題核心考點（第五節）

  • 練習題場景：35 歲女性，左足自大腿至小腿前側麻木。
  • 解題核心邏輯：
    1. 三（定位）：大腿至小腿前側跨越中、下焦。
    2. 二（辨經）：前側主要由陽明經所主。病在左，取右側。
    3. 一（找點）：在右側手部（或足部）尋找陽明經同氣點。因跨度大，可在右側手部合谷穴至曲池穴連線的對應比例位置尋找阿是穴。

21.6 解說：醫療實務建議

足部及下肢醫療實務，務必遵循以下資深建議：

  • 「阿是」的絕對精確度：足部骨骼肌肉結構厚實，患者疼痛往往非常明確。你們在對側同氣區（如腕踝或肘膝）尋找阿是點時，必須用拇指尖「地毯式」按壓。那個讓患者叫出聲來的酸痛點，就是氣血轉化的「開關」。
  • 守住「肘膝以下」的法脈：這是內針的靈魂。即便病在臀部或大腿，也絕不在該處下針。透過「陰陽倒換」將其映射到腕踝或肘膝，這不僅是為了安全，更是為了利用經絡遠端的強大調節力。
  • 導引的即時反饋：對於足部受傷或膝關節不利的患者，進針後一定要請他「試著下地走兩步」或「彎曲膝蓋」。導引配合針刺，能引導相火歸位，這也是內針最具魅力之處。
  • 多經合治的果斷：下肢痛症（如坐骨神經痛）往往同時涉及太陽、少陽甚至陽明。臨床上不要拘泥於一經，應根據疼痛的分佈，果斷進行多經合治。
