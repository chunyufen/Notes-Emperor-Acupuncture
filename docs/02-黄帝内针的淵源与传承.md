# 黄帝内针的淵源与传承

## 黄帝内针的淵源

黄帝内针代代皆为单传秘授。

- 杨其海先生自父杀杨运濟手中传得针灸一脉。
- 2016 年年底，由黄帝内针传人杨真海先生传讲、黄帝内针传承弟子刘力红先生整理的 《黄帝内针一一和平的使者》一书正式公开出版。

## 黄帝内针的传承

传承的路径大致有三条：

- 文字传承、
- 口耳传承和
- 直接传承。

传承要着重把握三个要点：

- 信（确信皇帝内针的理法方针）、
- 诚（按照孝悌忠信礼义廉耻的要求做好人）和
- 明（明针法背后的至深之理）。
