3  第三章：黃帝內針的基本特點

3.1 第一節：執兩用中

  • 主要概念與理論：
    • 核心奧妙：源自《中庸》與《論語》，意即「執其兩端，用其中於民」。
    • 兩端與中：在針灸臨床中，「兩端」即是左與右、上與下、陰與陽。
    • 操作邏輯：當病症表現在「左（一端）」時，醫者必須在「右（另一端）」尋求平衡。只有符合左右、上下對應的原則，才能實現「中（陰陽自和）」的目的。
  • 醫案示例要點：
    • 邏輯轉化：不論病症表現如何，只要能確定其在某一端，便在另一端施治。內針不追求繁複的補瀉手法，因為「中」的作用一旦被激發，人體會自動回歸天道的自然補瀉。
  • 練習題核心考點：
    • 理解「執兩用中」是內針臨床選穴的大原則：以右治左、以左治右。

3.2 第二節：隨證治之

  • 主要概念與理論：
    • 證（症）為眼目：內針強調「隨證治之」，而非隨「病名」或「脈象」治之。
    • 三位一體：所謂的「證」包含了病處（位）、病因與病治。它是機體對異常狀況的綜合表達。
  • 醫案示例要點：
    • 定經與定位：臨床診斷的第一步是確定「證」在哪個經絡（如陽明經或少陽經），以及在哪個部位（如頭部或胸部）。只有「證」定準了，後續的取穴才有方向。
  • 練習題核心考點：
    • 辨析「證」與「病」的區別。在內針體系中，症狀在哪經、哪位，即是治療的唯一指引。

3.3 第三節：同氣相求

  • 主要概念與理論：
    • 法理之魂：這是內針取穴定位的最高準則。
    • 同氣的類別：
      1. 同名經同氣：例如手陽明與足陽明同氣。
      2. 同位同氣（三焦同氣）：上部（天部）與上部同氣，中部（人部）與中部同氣，下部（地部）與下部同氣。
    • 核心口訣：「穴可失，經可離，同氣不能丟」。
  • 醫案示例要點：
    • 尋找同氣點：治病就是求同。若病在某一經、某一部位，施治的穴位必須在對應的同氣經絡與同氣部位上。若同氣不失，即便是偏離了標準穴位，也能見效。
  • 練習題核心考點：
    • 同氣相求的層次：理解「同名經同氣」與「三焦同氣」如何交互運用。

3.4 第四節：導引

  • 主要概念與理論：
    • 醫患共振：導引是內針「五術」之一，其本質是通過「感」來實現機體陰陽的自利與自和。
    • 身心貫通：針刺入後，醫者需「引」，患者需「感」。將患者的注意力從痛苦引向當下的變化，實現身心合一。
  • 醫療實務規範：
    • 留針時間：一般留針 45 分鐘。在此期間，患者應靜心留神，感受患處的細微變化。
  • 練習題核心考點：
    • 導引在五術中的地位及其實現代謝自利的深度意義。

3.5 第五節：神與形俱

  • 主要概念與理論：
    • 君火與相火：君火（心）以明，相火（心包、命門）以位。君火提供「明能」（智慧與光明），相火提供「熱能」（動能與變化）。
    • 守神與守形：上等的用針是以「神」為依據（守神），下等是以「形」為依據（守形）。內針追求從守形上升到守神的境界。
    • 阿是穴的真意：內針中的「阿是」即是精確的定位。它是為了確定相火系統的作用位置。
  • 進階解說：
    • 工在針外：內針的最高境界在於醫者的德行（大醫精誠）與對法理的把握。心法不可測，實則在於醫者的內修。
  • 練習題核心考點：
    • 理解「形」與「神」的關係，以及為什麼說「穴位的位定對與否，決定了相火效用的發揮」。

3.6 醫療實務的進階建議

醫療實務中，務必遵循以下原則：

  1. 初級階段：死守「同氣相求」與「左右治則」。不要糾結於複雜的病理，先看病在左還是在右，在哪一焦，屬哪一經。
  2. 中級階段：學會「導引」。不要把針紮進去就離開去喝茶。要與患者對話，引導他們去「找感覺」，這種醫患之間的能量流動是提升療效的秘訣。
  3. 進階階段：體悟「神與形俱」。意識到你手中的針不只是金屬，而是調整生命「相火」的工具。醫者的心定，針才靈。
